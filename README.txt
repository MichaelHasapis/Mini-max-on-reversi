Όσο αφορά την μεταγλώτισση καλούμε τον μεταγλωτιστή της java χωρίς όρισμα (javac ".java)
Όσο αφορά την εκτέλεση τρέχουμε το αρχείο Reversi από ένα command prompt(java Reversi)
Κατά την εκτέλεση του προγράμματος ζητάμε να δoθεί το max depth για τον MiniMax (θετικός ακέραιος αριθμός.
Έπειτα ζητάμε να επιλέξει ο παίκτης αν θα παίξει πρώτος ή μετά τον υπολογιστή (1 για να παίξει ο υπολογιστής 2 για να παίξει ο παίκτης)
Τέλος όταν ζητούνται συντεταγμένες θα πρέπει να δίνεται πρώτα η σειρά και μετά η στήλη
που επιθυμεί ο παίκτης να τοποθετήσει και να διαχωρίζονται οι δυο αριθμοί με κενό (πχ για την θέση 2η σειρά 3η στήλη να γραφτει ως 2 3)
